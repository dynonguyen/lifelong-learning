## Quick Reference

```bash
# Most useful Go commands
go run main.go           # Run program
go build                 # Compile package
go test ./...            # Run all tests
go test -v -race ./...   # Verbose with race detection
go mod init module/path  # Initialize module
go mod tidy              # Clean up dependencies
go get package@version   # Add dependency
go fmt ./...             # Format code
go vet ./...             # Static analysis
go doc package           # View documentation
go env                   # View environment
go version               # Go version
```

> **Good to know:** The Go community values simplicity and readability. When in doubt, follow Effective Go and the Go Proverbs. Join the Gophers Slack for real-time help and discussions.
