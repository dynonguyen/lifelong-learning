# Installation & Setup

## Install Python

**macOS (using Homebrew):**

```bash
brew install python@3.12
```

**Linux (Ubuntu/Debian):**

```bash
# Update package list
sudo apt update

# Install Python 3
sudo apt install python3 python3-pip python3-venv

# Verify installation
python3 --version
```

**Windows:**

Download the installer from [python.org/downloads](https://www.python.org/downloads/) and run it. Make sure to check "Add Python to PATH" during installation.

## Verify Installation

```bash
python3 --version
# Output: Python 3.12.0

pip3 --version
# Output: pip 23.3.1 from /usr/local/lib/python3.12/site-packages/pip (python 3.12)
```

## Create a Virtual Environment

Virtual environments isolate project dependencies:

```bash
# Create virtual environment
python3 -m venv myenv

# Activate on macOS/Linux
source myenv/bin/activate

# Activate on Windows
myenv\Scripts\activate

# Deactivate
deactivate
```

## Initialize a Python Project

```bash
# Create project directory
mkdir myproject && cd myproject

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Create requirements.txt
touch requirements.txt

# Create main file
touch main.py
```

## Create and Run Your First Program

```python
# main.py
def main():
    print("Hello, Python!")

if __name__ == "__main__":
    main()
```

```bash
# Run the program
python main.py

# Or make it executable (Unix-like systems)
chmod +x main.py
./main.py
```

## Install Packages

```bash
# Install a package
pip install requests

# Install from requirements.txt
pip install -r requirements.txt

# Save installed packages
pip freeze > requirements.txt

# Upgrade pip
pip install --upgrade pip
```

## Install Development Tools

```bash
# Linting and formatting
pip install black        # Code formatter
pip install flake8       # Linter
pip install mypy         # Type checker
pip install pylint       # Code analyzer

# Testing
pip install pytest       # Testing framework
pip install pytest-cov   # Coverage plugin

# Project management
pip install poetry       # Dependency management
pip install pipenv       # Virtual env + package management
```

## Set Up VS Code for Python

Install the official Python extension:

```bash
# Or install via VS Code
code --install-extension ms-python.python
```

Create `.vscode/settings.json`:

```json
{
	"python.defaultInterpreterPath": "${workspaceFolder}/venv/bin/python",
	"python.linting.enabled": true,
	"python.linting.pylintEnabled": true,
	"python.formatting.provider": "black",
	"editor.formatOnSave": true,
	"python.testing.pytestEnabled": true
}
```

> **Good to know:** Always use virtual environments to avoid conflicts between project dependencies. Use `python3` and `pip3` commands on systems where Python 2 might still be installed. The `if __name__ == "__main__":` idiom allows a file to be both imported as a module and run as a script.
