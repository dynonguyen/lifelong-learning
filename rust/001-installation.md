# Installation & Setup

## Install Rust with rustup

**macOS/Linux:**

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

**Windows:**

Download and run the installer from [rustup.rs](https://rustup.rs/)

**Verify Installation:**

```bash
rustc --version
cargo --version
```

## Update Rust

```bash
rustup update
```

## Install Specific Toolchain

```bash
# Install stable
rustup install stable

# Install nightly
rustup install nightly

# Set default
rustup default stable
```

## Create a New Project

**Binary (executable) project:**

```bash
cargo new my_project
cd my_project
```

**Library project:**

```bash
cargo new --lib my_library
cd my_library
```

## Build and Run

```bash
# Build the project
cargo build

# Build with optimizations
cargo build --release

# Run the project
cargo run

# Check if code compiles (faster than build)
cargo check
```

## Development Tools

**Install rust-analyzer (LSP for IDE support):**

Most IDEs have extensions for rust-analyzer. For VS Code:

- Install "rust-analyzer" extension

**Install rustfmt (code formatter):**

```bash
rustup component add rustfmt
```

**Install clippy (linter):**

```bash
rustup component add clippy
```

> **Good to know:** `rustup` is both the Rust installer and version manager. It manages multiple Rust versions and associated tools. Unlike Node.js or Python, Rust projects compile to native binaries without requiring a runtime.
